## 二、广播（共10分）

### 题目
已知广播接收者 `MyReceiver`，要求：
1. 动态注册，接收action为`zzuli`的广播（6分）
2. 按钮点击发送广播（4分）

### 答案

```java
public class ActivityA extends AppCompatActivity {
    private MyReceiver receiver;  // 声明成员变量

    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_a);
        
        // === 动态注册广播（6分）===
        receiver = new MyReceiver();                    // 1分：创建接收者
        IntentFilter filter = new IntentFilter("zzuli"); // 2分：设置过滤器，action为zzuli
        registerReceiver(receiver, filter);              // 3分：注册
    }

    @Override
    protected void onDestroy() {
        super.onDestroy();
        // 注销广播
        unregisterReceiver(receiver);  // 必须注销，否则内存泄漏
    }
    
    // === 发送广播（4分）===
    public void send(View view) {
        Intent intent = new Intent("zzuli");  // 2分：创建Intent，action为zzuli
        sendBroadcast(intent);                 // 2分：发送广播
    }
}
```

### 评分点
**注册部分（6分）**：
- 创建 MyReceiver 对象（1分）
- 创建 IntentFilter，action 设置为 "zzuli"（2分）
- 调用 registerReceiver（3分）

**发送部分（4分）**：
- 创建 Intent，action 为 "zzuli"（2分）
- 调用 sendBroadcast（2分）

**易失分点**：
- action 字符串前后不一致
- 忘记在 onDestroy 中 unregisterReceiver

## 三、网络访问（共35分）

### 题目
使用 HttpURLConnection 实现：
1. 发送POST请求（page:1，pagesize:10）（15分）
2. 流转字符串（10分）
3. 显示到TextView（10分）

### 答案

```java
public class ActivityB extends AppCompatActivity {
    private TextView textView;

    // === 3）Handler接收消息，更新UI（10分）===
    private Handler handler = new Handler() {
        @Override
        public void handleMessage(@NonNull Message msg) {
            String result = (String) msg.obj;         // 3分：取出数据
            textView.setText(result);                 // 3分：显示到TextView
        }  // Handler定义：4分
    };

    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_b);
        
        textView = findViewById(R.id.textView);  // 找控件
        
        // 网络请求必须在子线程
        new Thread(new Runnable() {
            @Override
            public void run() {
                try {
                    // === 1）发送POST请求（15分）===
                    URL url = new URL("https://www.tb.com/book/list");  // 2分
                    HttpURLConnection conn = (HttpURLConnection) url.openConnection();  // 2分
                    conn.setRequestMethod("POST");    // 2分
                    conn.setDoOutput(true);            // 2分：允许写数据
                    conn.setDoInput(true);             // 1分：允许读数据
                    
                    // 拼接参数
                    String params = "page=1&pagesize=10";  // 2分
                    OutputStream os = conn.getOutputStream();
                    os.write(params.getBytes("UTF-8"));    // 2分
                    os.close();                             // 1分
                    
                    // === 2）流转字符串（10分）===
                    InputStream is = conn.getInputStream();  // 2分
                    BufferedReader reader = new BufferedReader(
                        new InputStreamReader(is, "UTF-8"));  // 2分
                    StringBuilder sb = new StringBuilder();   // 1分
                    String line;
                    while ((line = reader.readLine()) != null) {  // 3分
                        sb.append(line);
                    }
                    reader.close();                           // 1分
                    String result = sb.toString();            // 1分
                    
                    // 发消息给主线程
                    Message msg = Message.obtain();
                    msg.obj = result;
                    handler.sendMessage(msg);
                    
                } catch (Exception e) {
                    e.printStackTrace();
                }
            }
        }).start();
    }
}
```

### 评分点

**1）POST请求（15分）**：
- 创建URL对象（2分）
- openConnection并强转（2分）
- setRequestMethod("POST")（2分）
- setDoOutput(true)（2分）
- setDoInput(true)（1分）
- 拼接参数字符串（2分）
- getOutputStream写参数（2分）
- close()（1分）

**2）流转字符串（10分）**：
- getInputStream（2分）
- BufferedReader + InputStreamReader（2分）
- StringBuilder（1分）
- while循环readLine（3分）
- close()（1分）
- toString()（1分）

**3）显示到TextView（10分）**：
- Handler定义（4分）
- handleMessage取数据（3分）
- setText显示（3分）

**易失分点**：
- 忘记 setDoOutput(true)
- 忘记编码 "UTF-8"
- 流没有 close()
- 没用 Handler，直接在子线程 setText（会崩溃）

## 四、列表显示数据（共35分）

### JSON格式
```json
{
    "code": 200,
    "message": "成功！",
    "result": {
        "list": [
            {"name": "string", "author": "string", "price": 0.0, "link": "string"}
        ],
        "total": 0
    }
}
```

### 答案

```java
public class ActivityC extends AppCompatActivity {
    private List<Book> books;
    private ListView listView;
    private String str = "...";
    
    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_c);
        
        listView = findViewById(R.id.lv_book);
        books = parseBooks(str);
        
        // === 3）设置适配器（5分）===
        BookAdapter adapter = new BookAdapter(this, books);  // 3分
        listView.setAdapter(adapter);                        // 2分
        
        // === 4）ListView点击事件（5分）===
        listView.setOnItemClickListener(new AdapterView.OnItemClickListener() {
            @Override
            public void onItemClick(AdapterView<?> parent, View view, int position, long id) {
                Book book = books.get(position);  // 2分：取出点击的书
                Intent intent = new Intent(ActivityC.this, BookActivity.class);  // 1分
                intent.putExtra("link", book.getLink());  // 1分：携带link
                startActivity(intent);  // 1分
            }
        });
    }

    // === 1）JSON解析（13分）===
    private List<Book> parseBooks(String str) {
        List<Book> list = new ArrayList<>();  // 1分
        try {
            JSONObject root = new JSONObject(str);  // 2分
            int code = root.getInt("code");         // 1分
            if (code == 200) {                      // 1分
                JSONObject result = root.getJSONObject("result");  // 2分
                JSONArray listArray = result.getJSONArray("list"); // 2分
                for (int i = 0; i < listArray.length(); i++) {  // 1分
                    JSONObject obj = listArray.getJSONObject(i);  // 1分
                    Book book = new Book();
                    book.setName(obj.getString("name"));
                    book.setAuthor(obj.getString("author"));
                    book.setPrice(obj.getDouble("price"));
                    book.setLink(obj.getString("link"));  // 以上4行共1分
                    list.add(book);  // 1分
                }
            }
        } catch (JSONException e) {
            e.printStackTrace();
        }
        return list;
    }
}

// === 2）适配器（12分）===
class BookAdapter extends BaseAdapter {
    private List<Book> books;
    private Context context;
    
    public BookAdapter(Context context, List<Book> books) {
        this.context = context;
        this.books = books;
    }
    
    @Override
    public int getCount() {  // 1分
        return books.size();
    }
    
    @Override
    public Object getItem(int position) {  // 1分
        return books.get(position);
    }
    
    @Override
    public long getItemId(int position) {  // 1分
        return position;
    }
    
    @Override
    public View getView(int position, View convertView, ViewGroup parent) {  // 9分
        ViewHolder holder;
        if (convertView == null) {  // 1分
            convertView = LayoutInflater.from(context)
                    .inflate(R.layout.item_book, parent, false);  // 2分
            holder = new ViewHolder();
            holder.tvName = convertView.findViewById(R.id.tv_name);
            holder.tvAuthor = convertView.findViewById(R.id.tv_author);
            holder.tvPrice = convertView.findViewById(R.id.tv_price);  // 以上3行共2分
            convertView.setTag(holder);  // 1分
        } else {
            holder = (ViewHolder) convertView.getTag();  // 1分
        }
        
        Book book = books.get(position);  // 1分
        holder.tvName.setText(book.getName());
        holder.tvAuthor.setText(book.getAuthor());
        holder.tvPrice.setText(String.valueOf(book.getPrice()));  // 以上3行共1分
        
        return convertView;
    }
    
    static class ViewHolder {
        TextView tvName, tvAuthor, tvPrice;
    }
}
```

### 评分点

**1）JSON解析（13分）**：
- new ArrayList（1分）
- new JSONObject(str)（2分）
- getInt("code")（1分）
- code==200判断（1分）
- getJSONObject("result")（2分）
- getJSONArray("list")（2分）
- for循环（1分）
- getJSONObject(i)（1分）
- 四个字段赋值（1分）
- add(book)（1分）

**2）适配器（12分）**：
- getCount: return books.size()（1分）
- getItem: return books.get(position)（1分）
- getItemId: return position（1分）
- getView 共9分：
  - if(convertView==null)（1分）
  - inflate加载布局（2分）
  - findViewById三个控件（2分）
  - setTag（1分）
  - getTag（1分）
  - books.get(position)（1分）
  - setText三行（1分）

**3）设置适配器（5分）**：
- new BookAdapter(this, books)（3分）
- setAdapter（2分）

**4）点击事件（5分）**：
- books.get(position)（2分）
- new Intent（1分）
- putExtra("link", ...)（1分）
- startActivity（1分）

**易失分点**：
- JSON 嵌套层级搞错（result → list）
- getView 里忘记 ViewHolder 模式
- price 是 double 类型，用 getDouble 不是 getString
- 点击事件用错参数（用 id 而不是 position）
