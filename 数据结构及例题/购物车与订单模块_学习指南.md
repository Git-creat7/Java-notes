### 1.2 数据流向

```
【购物车流程】
1. 用户切换到购物车Tab → onResume()被调用
2. fetchCart() 发起网络请求 → GET /api/v1/shop-cart
3. requestCart() 使用HttpURLConnection获取数据
4. applyCartResult() 解析JSON并更新UI
5. renderCart() 渲染购物车列表

【结算流程】
1. 用户勾选商品点击"去结算" → 跳转CheckoutActivity
2. readItems() 解析购物车传来的JSON数据
3. loadDefaultAddress() 加载默认收货地址
4. 用户点击"提交订单" → submitOrder()
5. 发送POST请求 → /api/v1/order
6. 支付成功后跳转到我的订单页面
```

### 2.2 网络请求实现（实训要求重点）

```java
// 第134-162行
private void fetchCart() {
    String token = getToken();
    
    // 未登录直接显示空购物车
    if (token.isEmpty()) {
        cartItems.clear();
        renderCart();
        return;
    }
    
    // 显示加载中
    renderLoading();
    
    // 在子线程中执行网络请求
    new Thread(() -> {
        // 尝试多个baseUrl，直到成功
        for (String baseUrl : API_BASE_URLS) {
            try {
                List<CartItem> result = requestCart(baseUrl, token);
                currentBaseUrl = baseUrl; // 记录成功的URL
                
                // 切回主线程更新UI
                mainHandler.post(() -> {
                    if (!isAdded()) return; // 防止Fragment已销毁
                    applyCartResult(result);
                });
                return; // 成功则退出
            } catch (Exception ignored) {
                // 失败继续尝试下一个URL
            }
        }
        
        // 所有URL都失败，显示错误
        mainHandler.post(() -> {
            if (!isAdded()) return;
            cartItems.clear();
            renderCart();
            Toast.makeText(requireContext(), "购物车数据加载失败", Toast.LENGTH_SHORT).show();
        });
    }).start();
}
```

**关键点**：
1. ✅ 未登录处理：直接显示空购物车
2. ✅ 子线程请求：避免阻塞UI
3. ✅ 多URL尝试：提高连接成功率
4. ✅ 主线程更新UI：使用mainHandler.post()
5. ✅ 防止崩溃：检查isAdded()

#### ② requestCart() - HttpURLConnection实现（实训要求）
```java
// 第175-206行
private List<CartItem> requestCart(String baseUrl, String token) throws Exception {
    HttpURLConnection cn = null;
    try {
        // 1. 创建连接
        cn = (HttpURLConnection) new URL(baseUrl + "/api/v1/shop-cart").openConnection();
        cn.setRequestMethod("GET");
        cn.setConnectTimeout(2500);  // 连接超时2.5秒
        cn.setReadTimeout(3500);     // 读取超时3.5秒
        
        // 2. 设置Token（实训要求的方式）
        cn.setRequestProperty("token", token);
        
        // 3. 检查响应码
        if (cn.getResponseCode() != 200) 
            throw new IllegalStateException("bad response");
        
        // 4. 读取响应
        String text = NetworkHelper.readStream(cn.getInputStream());
        
        // 5. 解析JSON
        JSONObject obj = new JSONObject(text);
        if (obj.optInt("resultCode") != 200) 
            throw new IllegalStateException(obj.optString("message"));
        
        // 6. 构建购物车列表
        JSONArray data = obj.optJSONArray("data");
        List<CartItem> result = new ArrayList<>();
        for (int i = 0; data != null && i < data.length(); i++) {
            JSONObject item = data.optJSONObject(i);
            if (item == null) continue;
            
            CartItem cartItem = new CartItem();
            cartItem.cartItemId = item.optLong("cartItemId");
            cartItem.goodsId = item.optLong("goodsId");
            cartItem.goodsName = item.optString("goodsName", "新蜂精选商品");
            cartItem.goodsCount = Math.max(1, item.optInt("goodsCount", 1)); // 最小为1
            cartItem.sellingPrice = item.optInt("sellingPrice", 0);
            cartItem.goodsCoverImg = normalizeImageUrl(item.optString("goodsCoverImg", ""));
            result.add(cartItem);
        }
        return result;
    } finally {
        NetworkHelper.disconnect(cn); // 确保连接被关闭
    }
}
```

**实训要求符合点**：
1. ✅ 使用HttpURLConnection（不用OkHttp等第三方库）
2. ✅ Token使用方式：`cn.setRequestProperty("token", token)`
3. ✅ 设置超时时间
4. ✅ finally块关闭连接

```java
private void updateCartCount(CartItem item, int newCount) {
    new Thread(() -> {
        try {
            // 1. 构建请求体
            JSONObject body = new JSONObject();
            body.put("cartItemId", item.cartItemId);
            body.put("goodsCount", newCount);
            
            // 2. 发送PUT请求
            HttpURLConnection cn = (HttpURLConnection) 
                new URL(currentBaseUrl + "/api/v1/shop-cart").openConnection();
            cn.setRequestMethod("PUT");
            cn.setDoOutput(true);
            cn.setRequestProperty("Content-Type", "application/json");
            cn.setRequestProperty("token", getToken());
            
            // 3. 写入请求体
            OutputStream os = cn.getOutputStream();
            os.write(body.toString().getBytes(StandardCharsets.UTF_8));
            os.flush();
            os.close();
            
            // 4. 检查响应
            if (cn.getResponseCode() == 200) {
                String text = NetworkHelper.readStream(cn.getInputStream());
                JSONObject obj = new JSONObject(text);
                if (obj.optInt("resultCode") == 200) {
                    // 5. 成功后刷新购物车
                    mainHandler.post(() -> {
                        if (!isAdded()) return;
                        fetchCart();
                    });
                }
            }
        } catch (Exception e) {
            mainHandler.post(() -> {
                if (!isAdded()) return;
                Toast.makeText(requireContext(), "修改数量失败", Toast.LENGTH_SHORT).show();
            });
        }
    }).start();
}
```

#### goToCheckout() - 跳转到结算页
```java
private void goToCheckout() {
    // 1. 获取选中的商品
    List<CartItem> selected = new ArrayList<>();
    for (CartItem item : cartItems) {
        if (item.selected) selected.add(item);
    }
    
    // 2. 检查是否有选中商品
    if (selected.isEmpty()) {
        Toast.makeText(requireContext(), "请选择要结算的商品", Toast.LENGTH_SHORT).show();
        return;
    }
    
    // 3. 将商品数据序列化为JSON
    JSONArray jsonArray = new JSONArray();
    for (CartItem item : selected) {
        try {
            JSONObject obj = new JSONObject();
            obj.put("cartItemId", item.cartItemId);
            obj.put("goodsId", item.goodsId);
            obj.put("goodsName", item.goodsName);
            obj.put("goodsCoverImg", item.goodsCoverImg);
            obj.put("goodsCount", item.goodsCount);
            obj.put("sellingPrice", item.sellingPrice);
            jsonArray.put(obj);
        } catch (Exception e) {
            e.printStackTrace();
        }
    }
    
    // 4. 跳转到CheckoutActivity
    Intent intent = new Intent(requireContext(), CheckoutActivity.class);
    intent.putExtra(CheckoutActivity.EXTRA_CART_ITEMS, jsonArray.toString());
    startActivity(intent);
}
```

**数据传递方式**：
- 使用Intent传递JSON字符串
- CheckoutActivity在readItems()中解析

```java
// 第114-137行
private void readItems() {
    String raw = getIntent().getStringExtra(EXTRA_CART_ITEMS);
    try {
        JSONArray array = new JSONArray(raw == null ? "[]" : raw);
        for (int i = 0; i < array.length(); i++) {
            JSONObject obj = array.optJSONObject(i);
            if (obj == null) continue;
            
            CheckoutItem item = new CheckoutItem();
            item.cartItemId = obj.optLong("cartItemId");
            item.goodsId = obj.optLong("goodsId");
            item.goodsName = obj.optString("goodsName", "新蜂精选商品");
            item.goodsCount = Math.max(1, obj.optInt("goodsCount", 1)); // 保证≥1
            item.sellingPrice = obj.optInt("sellingPrice", 0);
            item.directBuy = obj.optBoolean("directBuy", item.cartItemId <= 0);
            items.add(item);
        }
    } catch (Exception ignored) {
        // ⚠️ 注意：这里应该添加错误处理
    }
}
```

**关键点**：
- 使用optXXX方法安全解析JSON
- Math.max(1, count) 保证数量最小为1
- directBuy判断：cartItemId<=0表示立即购买

```java
// 第139-152行
private void loadDefaultAddress() {
    try {
        String raw = getSharedPreferences("info", MODE_PRIVATE)
            .getString(defaultAddressJsonKey(), "");
        
        if (raw == null || raw.trim().isEmpty()) {
            // 兼容旧版数据
            raw = getSharedPreferences("info", MODE_PRIVATE)
                .getString("defaultAddressJson", "");
            if (raw != null && !raw.trim().isEmpty()) {
                getSharedPreferences("info", MODE_PRIVATE).edit()
                    .putString(defaultAddressJsonKey(), raw).apply();
            }
        }
        
        selectedAddress = (raw == null || raw.trim().isEmpty()) 
            ? null 
            : new JSONObject(raw);
    } catch (Exception e) {
        selectedAddress = null;
    }
}
```

**为什么这样做？**
- 按用户存储地址（多用户切换时不混淆）
- defaultAddressJsonKey() = "defaultAddressJson_用户名"

### 3.3 提交订单实现（核心）

#### submitOrder() - 提交订单
```java
private void submitOrder() {
    // 1. 检查是否选择了地址
    if (selectedAddress == null) {
        Toast.makeText(this, "请先选择收货地址", Toast.LENGTH_SHORT).show();
        return;
    }
    
    // 2. 禁用提交按钮，防止重复提交
    submitButton.setEnabled(false);
    submitButton.setText("提交中...");
    
    new Thread(() -> {
        try {
            // 3. 构建订单数据
            JSONObject requestBody = new JSONObject();
            
            // 3.1 添加商品ID列表
            JSONArray cartItemIds = new JSONArray();
            for (CheckoutItem item : items) {
                if (!item.directBuy) { // 购物车商品
                    cartItemIds.put(item.cartItemId);
                }
            }
            if (cartItemIds.length() > 0) {
                requestBody.put("cartItemIds", cartItemIds);
            }
            
            // 3.2 添加地址信息
            JSONObject address = new JSONObject();
            address.put("userName", selectedAddress.optString("userName"));
            address.put("userPhone", selectedAddress.optString("userPhone"));
            address.put("provinceName", selectedAddress.optString("provinceName"));
            address.put("cityName", selectedAddress.optString("cityName"));
            address.put("regionName", selectedAddress.optString("regionName"));
            address.put("detailAddress", selectedAddress.optString("detailAddress"));
            requestBody.put("address", address);
            
            // 3.3 添加优惠券（如果有）
            if (!selectedCoupon.isEmpty()) {
                requestBody.put("couponName", selectedCoupon);
            }
            
            // 4. 发送POST请求
            HttpURLConnection cn = (HttpURLConnection) 
                new URL(API_BASE_URLS[0] + "/api/v1/order").openConnection();
            cn.setRequestMethod("POST");
            cn.setDoOutput(true);
            cn.setRequestProperty("Content-Type", "application/json");
            cn.setRequestProperty("token", getToken());
            
            OutputStream os = cn.getOutputStream();
            os.write(requestBody.toString().getBytes(StandardCharsets.UTF_8));
            os.flush();
            os.close();
            
            // 5. 解析响应
            if (cn.getResponseCode() == 200) {
                String text = NetworkHelper.readStream(cn.getInputStream());
                JSONObject obj = new JSONObject(text);
                
                if (obj.optInt("resultCode") == 200) {
                    String orderNo = obj.optString("data");
                    
                    // 6. 跳转到支付页面
                    handler.post(() -> {
                        showPaymentDialog(orderNo);
                    });
                } else {
                    handler.post(() -> {
                        submitButton.setEnabled(true);
                        submitButton.setText("提交订单");
                        Toast.makeText(this, obj.optString("message"), Toast.LENGTH_SHORT).show();
                    });
                }
            }
        } catch (Exception e) {
            handler.post(() -> {
                submitButton.setEnabled(true);
                submitButton.setText("提交订单");
                Toast.makeText(this, "订单提交失败", Toast.LENGTH_SHORT).show();
            });
        }
    }).start();
}
```

```java
private void processPayment(String orderNo, int payType) {
    new Thread(() -> {
        try {
            // 1. 构建支付请求
            JSONObject body = new JSONObject();
            body.put("payType", payType); // 1=支付宝，2=微信
            
            // 2. 发送PUT请求
            HttpURLConnection cn = (HttpURLConnection) 
                new URL(API_BASE_URLS[0] + "/api/v1/order/" + orderNo + "/pay")
                    .openConnection();
            cn.setRequestMethod("PUT");
            cn.setDoOutput(true);
            cn.setRequestProperty("Content-Type", "application/json");
            cn.setRequestProperty("token", getToken());
            
            OutputStream os = cn.getOutputStream();
            os.write(body.toString().getBytes(StandardCharsets.UTF_8));
            os.flush();
            os.close();
            
            // 3. 检查支付结果
            if (cn.getResponseCode() == 200) {
                String text = NetworkHelper.readStream(cn.getInputStream());
                JSONObject obj = new JSONObject(text);
                
                if (obj.optInt("resultCode") == 200) {
                    // 4. 支付成功，跳转到订单页面
                    handler.post(() -> {
                        Toast.makeText(this, "支付成功", Toast.LENGTH_SHORT).show();
                        
                        // 跳转到我的订单页面
                        Intent intent = new Intent(this, MyOrdersActivity.class);
                        intent.setFlags(Intent.FLAG_ACTIVITY_CLEAR_TOP);
                        startActivity(intent);
                        finish();
                    });
                } else {
                    handler.post(() -> {
                        Toast.makeText(this, "支付失败：" + obj.optString("message"), 
                            Toast.LENGTH_SHORT).show();
                    });
                }
            }
        } catch (Exception e) {
            handler.post(() -> {
                Toast.makeText(this, "支付失败", Toast.LENGTH_SHORT).show();
            });
        }
    }).start();
}
```

### 4.2 常见答辩问题

#### Q1：购物车数据如何刷新？
**A**：
1. onResume()中调用fetchCart()
2. 数量修改成功后调用fetchCart()

#### Q2：如何防止内存泄漏？
**A**：
```java
@Override
public void onDestroyView() {
    super.onDestroyView();
    mainHandler.removeCallbacksAndMessages(null); // 清除Handler消息
}
```
Handler持有Fragment/Activity引用，不清理会导致内存泄漏。

---

#### Q3：未登录状态如何处理？
**A**：
- **购物车**：token为空时直接显示空购物车或登录提示
- **订单页**：onCreate中调用ensureLogin()，未登录跳转到LoginActivity并finish()
