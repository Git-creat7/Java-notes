# 数据库期末复习整理目录（SQL Server）

建议使用顺序：

1. 先看 `02_SQL题模板与往年题改写_SQLServer.md`：优先拿 SQL 30 分。
2. 再看 `03_综合题答题模板.md`：优先拿综合题 40 分。
3. 最后看 `01_数据库期末考前速记.md`：补填空和简答 30 分。

考前最后一小时：

- 20 分钟：背 SQL Server 常用模板，尤其 JOIN、GROUP BY、子查询、UPDATE FROM、GRANT/REVOKE。
- 25 分钟：背 ER 转表、主外键、2NF/3NF、事务并发恢复安全模板。
- 15 分钟：快速过一遍概念速记中的高频关键词。

注意：本文档均按 SQL Server / T-SQL 口径整理，避免使用 MySQL 的 `AUTO_INCREMENT`、`LIMIT`、`UPDATE ... JOIN ... SET` 等写法。
